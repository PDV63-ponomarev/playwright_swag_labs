1. Авторизация 
- Вход с валидными данными
- Вход с невалидными данными
- Log out

2. Каталог товаров
- Отображение списка товаров (изображение, цена, название, рейтинг)
- Сортировка (по цене, популярности, новизне)
- Фильтрация (по категориям, брендам, цене, размеру и т.д.)
- Карточка товара (фото, название, описание, количество, кнопка «В корзину»)

3. Поиск
- Поиск по существующему слову
- Поиск по несуществующему слову
- Поиск с опечатками (если есть подсказки)
- Пустой поиск

4. Корзина
- Добавление товара (с выбором количества, размера, цвета)
- Изменение количества в корзине
- Удаление товара из корзины
- Переход к оформлению заказа
