В данном проекте представлены демонстрационные автотесты.  
Тесты написанный на **Playwright** Python.  
Тесты запускаются в **GitHub Actions** и локально.  
Отчет демонстрация сформирован в [**Allure**](https://pdv63-ponomarev.github.io/playwright_swag_labs/23/index.html)

### Автотесты для [**_Swag Labs_**](https://www.saucedemo.com):
<details> <summary>1. Авторизация </summary>    

- Вход с валидными данными  
- Log out
- Вход с неверным логином
- Вход с пустым логином
- Вход с пустым паролем
- Вход с заблокированным пользователем
</details>
<details>
<summary>2. Каталог товаров</summary>

- Отображение списка товаров (изображение, название, описание, цена)
- Сортировка (по алфавиту, цене)
- Карточка товара (фото, название, описание, цена, кнопка «В корзину»)
</details>

<details>
<summary>3. Корзина</summary>  

- Добавление товара 
- Проверка товара в корзине
- Удаление товара из корзины
- Переход к оформлению заказа
</details>
